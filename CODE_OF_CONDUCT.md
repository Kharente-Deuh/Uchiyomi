# Code of Conduct

This project adopts the [Contributor Covenant][homepage], version 2.1,
available at
https://www.contributor-covenant.org/version/2/1/code_of_conduct.html.

In short: be respectful, welcoming, and constructive. Harassment and
exclusionary behavior are not tolerated.

Report unacceptable behavior to the maintainers via the contact path in
[`SECURITY.md`](./SECURITY.md) or a private channel.

[homepage]: https://www.contributor-covenant.org
